Need to download enchant to run text processing. Run 'sudo pip install pyenchant' for unix. If you are using MacOSX, unzip the pyenchant egg in required_pkgs folder and copy folder enchant to your Python site-packages folder. 

First time using nltk, import nltk and run nltk.download() in python console to download all the required packages. Required packages are 'stopwords', 'nps_chat', 'wordnet', and 'words' from corpus and 'maxent_treebank_pos_tagger' from all packages.
